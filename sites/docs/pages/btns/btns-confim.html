通用参数说明

参数 说明 类型 必填 备注
defaultMode 类型 string confirm（确定按钮）
functionClass 类名 string modal/list
isCloseWindow 是否打开页面 boolean
type 传参类型 string> ajax类型，GET/POST (默认GET)
dataFormat string 两个值 id object
dataSrc 返回的值 string 两个值 data/rows
text 文本 string
title 标题 string>
ajaxData data入参值 object
requestSource 发送形式 string selected（vo形式）checkbox（数组格式）thisVo（全部）none（无）
isMainDbAction 是否执行双击事件 boolean
isInlineBtn 是否行内按钮 boolean
isMobileInlineBtn 是否手机行内按钮 boolean
disabledExpression 禁用表达式 string {1:'disabled'}
parameterFormat 参数格式化 string>
validateParams 验证参数 object 只验证指定参数
successMsg 验证提示语 string
successOperate 对页面进行 string refresh(刷新) delete(删除) edit(编辑) add(添加)
matrixVariable 矩阵传值 object
isSendPageParams 是否发送页面参数 boolean
shortcutKey 快捷键 string
isKeepSelected 是否行选中 boolean
