参数说明

参数 说明 类型 必填 备注
id 唯一标识 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string checkbox
textField 显示字段 string 选项显示的字段(默认text)
valueField 保存值 string 选项value字段，即如果string类型（isObjectValue==false）组件的保存值 (默认value)
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string 当前组件仅支持必填验证：required
value 显示的值 string 默认显示的值
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度 与width不共存(默认100)
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
acts 静态参数 string 静态模式时的参数 详见：静态数据配置
subdata 数据 array 选项数组 具体配置见下方：subdata配置 数据类型[]
url 获取地址 string 通过ajax获得subdata
dataSrc 返回的值 string ajax返回数据的key值
type 类型 string ajax类型，GET/POST (默认GET)
data 传参 object ajax传参 关联字段配置（即获得表单中其它字段值作为传输参数）{}
contentType ajax的传值类型 string ajax的传值类型 ‘application/json’表示传值data转化成字符串
relationField 关联字段 string 关联字段 即该字段发生变化relationField对应的字段发生变化
isObjectValue 显示值格式 boolean value值的格式(默认false)
outputFields 获取值 boolean 获取值
formSource 组件模式 string 组件模式；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认from)
templateName 组件模式名字 string 组件模板名字；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认pc)
mindjetFieldPosition 是否在更多种显示 string 是否是更多中显示的数据(默认field)
isBooleanValue boolean string value是否是0/1(默认false)

显示的值

显示默认选择 
    {
        id: 'value',
        label: '显示 :',   
        type: 'checkbox',  
        value: '1', //value为1时默认显示男
        textField: 'name',
        valueField: 'id',
        subdata:[
            {id:'1', name:'男', state:""},
            {id:'2', name:'女'},
            {id:'3', name:'其它'}
        ],
        outputFields:{
            name:"{name}",
            statesss : "{state}",
        }
    }

获取数据1

用subdata获取数据 
        {
            id: 'subdata',
            label: 'subdata传出:',
            type: 'checkbox',
            value: "1",
            textField: 'name',
            valueField: 'id',
            subdata:[               //通过subdata获取值 
                {id:'1', name:'男'},

                {id:'3', name:'其它'}
            ]
        }

获取数据2

用ajax获取数据 
        {
            id: 'ajax',
            label: 'ajax传出:',
            type: 'checkbox',
            value: "1",
            textField: 'name',
            valueField: 'id',
            ajaxConfig:{
                url: '/docs/data/select/radio.json' ,
                type: 'GET',
                contentType: 'application/x-www-form-urlencoded', 
                dataSrc: 'rows',
            },
            
        }

复选框参数说明

参数 类型 必填 备注
textField 显示值 string 显示字段，没有会组件出错，选项都显示undefined
valueField 保存值 string 保存字段，没有组件保存值出错
isChecked 是否选中 boolean 是否选中 没有表示不选中
isDisabled 是否只读 boolean 是否只读，若果组件配置的disabled==true那么该值不起作用，左右选项设置只读

复选框使用注意

  • subdata和url配置一个即可，都配置时会通过ajax返回值替换掉配置的subdata，即url优先级高
  • subdata和url必须有一个配置，否则没有选项
  • disabled的优先级高于subdata中isDisabled的优先级
  • value的优先级高于subdata中isChecked的优先级
  • data中this.*,*对应的字段表单中必须存在
  • url配置方式可以是以下格式:
    ajaxConfig : {
        url ： "",
        type: "",
        dataSrc:"",
        contentType:'',
        data:  {}，
    }
