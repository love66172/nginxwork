参数说明

参数 说明 类型 是否必填 备注
id 唯一标识 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
source 获得 object 弹框调取的地址 参数：url：‘’ 默认{}
searth 路径 object 回车搜索配置 参数：url：‘’；dataSrc：‘’；默认{}
type 组件类型 string business
textField 显示字段 string 选项显示的字段(默认text)
dialogTitle 弹框标题 string 弹框标题(默认默认弹框标题)
infoBtnName 基本信息 string 基本信息按钮名字
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string 当前组件仅支持必填验证：required
value 显示的值 string 默认显示的值
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度 与width不共存(默认100)
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
acts 静态参数 string 静态模式时的参数 详见：静态数据配置
isShowDialog 业务组件显示 boolean 当前业务组件的弹框是否正在显示(默认false)
isReadOutputFields 代码配置 string 是否配置outputFields(false)
defaultPageIndex 默认显示页码 number 默认false
isOutputString 是否输出字符串 string 通过弹框获取textField获取value(默认value)

显示的值

显示默认选择 

    {
        id: 'value',
        label: '显示 :',   
        type: 'business',  
        value: '1', //value为1时默认显示男
        textField: 'name',
        valueField: 'id',
        subdata:[
            {id:'1', name:'男', state:"none"},
            {id:'2', name:'女'},
            {id:'3', name:'其它'}
        ],
        outputFields:{
            name:"{name}",
            statesss : "{state}",
        }
    }
