参数说明

参数 说明 类型 必填 备注
id 唯一标识 string 组件id，实际生成的表单中名称为'form-'+id id不可重复
type 组件类型 string date
label 标签 string 组件的标签，未配置则不显示
rules 验证规则 string 当前组件仅支持 必填验证：required
value 显示的值 string 默认显示的值
disabled 只读 boolean 是否只读(默认flase)
inputWidth 宽 number 组件除去label的宽度 与width不共存(默认100)
width 宽 string 组件整体宽度 *% 与inputWidth不共存
changeHandler 变更处理程序 function 组件值改变时触发 即value值发生改变时触发
hidden 隐藏 boolean 是否隐藏(默认flase)
acts 静态参数 string 静态模式时的参数 详见：静态数据配置
formSource 组件模式 string 组件模式；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认from)
templateName 组件模式名字 string 组件模板名字；直接调用组件时配置，表单配置时不需要配置直接通过表单配置设置了组件配置(默认pc)
mindjetFieldPosition 是否在更多种显示 string 是否是更多中显示的数据(默认field) field-more
startView 开始视图 number 开始视图 0(日)/1(月->日)/2(年->月->日)(默认from)
addvalue 添加按钮 object 日期底部按钮 配置 {value:''，id:''}(默认pc)
format 日期格式 string 日期格式 yyy-mm-dd (默认：nsVals.default.dateFormat)
isInputMask 样式验证 boolean 输入样式验证
isDefaultDate 默认日期 boolean 是否设置默认日期(默认false)
isTime 添加时间 boolean 是否添加时间
daysOfWeekDisabled 字符串或数组格式 string 字符串或数组格式 根据周几设置只读 默认''即不设置只读

显示的值

默认显示内容:日期组件value值必须是数字 
    {
        id: 'value',
        label: '显示 :',
        type: 'date',
        value: 1544148749870,  //value值必须是数字
    }

只读

只读，不可编辑 
    {
        id: 'disabled',
        label: '只读 :',
        type: 'date',
        disabled: true,   //只读模式
    }

样式宽

宽 
    {
                
        id: 'width',
        label: 'width宽:',
        type: 'date',
        iwidth: 300,    //宽

    }

验证

验证规则 
    {
        id: 'requied',
        label: '只读 :',
        type: 'map',
        rules:'requied',   //必填验证                    
    }

是否显示时间

显示时间 
    {
        isHasClose: false,
        id: 'isTime',
        label: '显示时间:',
        type: 'date',
        isTime:true, //是否显示时间
    }

添加按钮

添加日期按钮 
    {
        isHasClose: false,
        id: 'addvalue',
        label: '添加按钮:',
        type: 'date',
        value: 1544148749870,
        addvalue:{          //添加按钮
            value:'添加按钮',
            id:'1',
        },
    }

默认日期

是否设置默认日期 
    {
        id: 'isdefaultdate',
        label: '默认日期 :',
        type: 'date',
        isDefaultDate:true, //设置默认日期
    }

开始视图

日期开始视图 
    {

        type: 'date',
        value: 1544148749870,

    }

日期格式

日期格式 9数字 a字母 y年 h小时 s年 m月 
    {
        id: 'date1',
        label: '生日1',
        type: 'date',
        value: 1564761383000,
        format: 'HH:mm:ss',

    }
